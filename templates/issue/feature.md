# Feature Template

## Structure

```markdown
## Summary

[Brief overview of the feature]

## Acceptance Criteria

- [ ] [Observable behavior 1]
- [ ] [Observable behavior 2]
- [ ] [Observable behavior 3]

## Scope

- **In scope**: [What this feature includes]
- **Out of scope**: [What this feature excludes]

## Notes (optional)

[Additional context, links, or considerations]
```

## Guidelines

| Field               | Description                                                 |
| ------------------- | ----------------------------------------------------------- |
| Summary             | What this feature does in 1-2 sentences                     |
| Acceptance Criteria | Observable, testable behaviors (not implementation details) |
| Scope               | Clear boundaries to prevent scope creep                     |
