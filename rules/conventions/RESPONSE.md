# Response

## Rules

| Rule             | Description                                                                                                                          |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| Conclusion first | Lead with the conclusion or result. Context and reasoning follow                                                                     |
| Recommend first  | Recommend → weakness → reasoning → alternatives. Disclosing the weakness prevents "AI recommended it, so it must be right" thinking. |
| Declare then act | Announce what you will do in one line before acting                                                                                  |
| Ask concisely    | When seeking a decision, ask directly without lengthy explanation                                                                    |
