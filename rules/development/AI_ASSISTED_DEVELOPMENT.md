# AI-Assisted Development

**Mindset**: AI generates, humans validate

## Project Rules

1. **Review is mandatory** - Never commit AI code without understanding it
2. **Past examples > abstract rules** - Reference existing code as examples
3. **You own the code** - AI is assistant, not replacement

## Integration

| Principle               | AI Role                    | Human Role        |
| ----------------------- | -------------------------- | ----------------- |
| TDD                     | Generate test scaffolding  | Write assertions  |
| Occam's Razor           | Question every abstraction | Apply simplicity  |
| Progressive Enhancement | Generate simple version    | Enhance as needed |
