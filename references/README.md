# References

Auxiliary documents referenced by commands and skills.

**Note**: Files in this directory are NOT loaded as slash commands.

## Structure

```text
references/
└── commands/           # Documents referenced by commands
    ├── code/           # For /code command
    ├── fix/            # For /fix command
    └── shared/         # Shared across multiple commands
```

## Usage

Reference from commands or skills:

```markdown
[@../references/commands/code/quality-gates.md]
```

## Future Extensions

Add as needed:

- `references/skills/` - Documents for skills
- `references/agents/` - Documents for agents
- `references/rules/` - Theoretical/principle documents
