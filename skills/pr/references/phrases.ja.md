# 空句 (日本語)

PR 本文に出やすい LLM 的な空句を検出して直す。本文が日本語のとき prose-review.md と併せて使う。

| パターン         | シグナル                                              | 修正                                 |
| ---------------- | ----------------------------------------------------- | ------------------------------------ |
| 定型の書き出し   | 「この PR は〜を導入する / 実装する / 追加する」      | 解決した問題やアウトカムから始める   |
| 空虚な強調語     | 包括的 / 抜本的 / シームレス / 徹底的                 | 削除するか具体 (件数、名前) に置換   |
| 埋め草の動詞     | 活用する / 利用する / 容易にする                      | 「使う」「する」など平易な動詞にする |
| あいまいな数量詞 | さまざまな変更 / 複数の改善 / いくつかの修正          | 列挙するか件数を示す                 |
| ヘッジの重ね掛け | おそらく〜かもしれない / 〜の可能性があるかもしれない | hedge は最大 1 つ、または断言する    |
| 埋め草フレーズ   | 「ご意見をお待ちしています」「フィードバック歓迎」    | 削除。事実を述べるか直接尋ねる       |
