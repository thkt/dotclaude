| セクション       | 説明                                   |
| ---------------- | -------------------------------------- |
| 用語集           | ドメイン固有の用語と定義               |

| 関連             | エンティティ間の関係を示すER図         |
| ビジネスルール   | ドメインルールとその適用箇所           |
| ドメインイベント | イベントとその購読者                   |
