# メンテナンステンプレート

## 構造

```markdown
## 概要

[必要なメンテナンス作業]

## タイプ

- [ ] 依存関係の更新
- [ ] 設定変更
- [ ] コード整理 / リファクタ
- [ ] CI/CD改善
- [ ] その他メンテナンス

## 詳細

[具体的な変更内容]

## 影響範囲（任意）

[この変更で影響を受ける可能性のある箇所]
```

## ガイドライン

| フィールド | 説明                         |
| ---------- | ---------------------------- |
| 概要       | 必要なメンテナンス作業       |
| タイプ     | メンテナンス作業のカテゴリ   |
| 詳細       | 具体的な変更内容             |
| 影響範囲   | 影響を受ける可能性のある箇所 |
