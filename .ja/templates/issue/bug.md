# バグレポートテンプレート

## 構造

```markdown
## 概要

[バグの1行説明]

## 再現手順

1. [ステップ1]
2. [ステップ2]
3. [ステップ3]

## 期待 vs 実際

- **期待**: [期待される動作]
- **実際**: [実際の動作]

## 環境（該当する場合）

- ブラウザ/OS: [例: Chrome 120 / macOS 14]
- バージョン: [例: v1.2.3]
```

## ガイドライン

| フィールド | 説明                                 |
| ---------- | ------------------------------------ |
| 概要       | 1文、具体的、アクション可能          |
| 手順       | 番号付き、最小限の再現ステップ       |
| 期待/実際  | 意図した動作と現在の動作の明確な対比 |
| 環境       | 環境固有のバグの場合に記載           |
