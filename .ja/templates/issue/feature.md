# 機能テンプレート

## 構造

```markdown
## 概要

[機能の簡単な説明]

## 受け入れ基準

- [ ] [観察可能な動作1]
- [ ] [観察可能な動作2]
- [ ] [観察可能な動作3]

## スコープ

- **スコープ内**: [この機能に含まれるもの]
- **スコープ外**: [この機能に含まれないもの]

## 備考（任意）

[追加のコンテキスト、リンク、考慮事項]
```

## ガイドライン

| フィールド   | 説明                                           |
| ------------ | ---------------------------------------------- |
| 概要         | この機能が何をするか1-2文で                    |
| 受け入れ基準 | 観察可能でテスト可能な動作（実装詳細ではなく） |
| スコープ     | スコープクリープを防ぐ明確な境界               |
