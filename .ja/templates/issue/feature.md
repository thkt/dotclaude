# 機能テンプレート

## 構造

```markdown
## What & Why

[何を作るか — 1-2文]
[なぜ必要か — ユーザー課題やビジネス理由]

## 受け入れ基準

- [ ] [○○すると、△△になる]
- [ ] [○○すると、△△になる]

## スコープ

- **スコープ内**: [この Issue で対応すること]
- **スコープ外**: [この Issue では対応しないこと]

## 制約（任意）

- [技術制約、禁止アプローチ、依存関係]
```

## ガイドライン

| フィールド   | OK                                           | NG                              |
| ------------ | -------------------------------------------- | ------------------------------- |
| What & Why   | 「CSV出力を追加し、オフライン分析を可能に」  | 「CSV出力を追加」（Whyなし）     |
| 受け入れ基準 | 「エクスポートを押すと .csv がDLされる」      | 「CSV出力が正しく動作する」      |
| スコープ外   | 「Excel形式はスコープ外」                     | （省略）                         |
| 制約         | 「新規依存の追加禁止」                        | （制約があるのに省略）           |
