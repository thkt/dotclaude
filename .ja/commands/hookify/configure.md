# /hookify:configure - フックルール設定

フックルールの有効化、無効化、または削除をインタラクティブに実行。

## プロセス

1. `/hookify:list`でルールを**一覧表示**
2. 設定するルールをユーザーに**確認**
3. **オプションを表示**:
   - ルールを有効化
   - ルールを無効化
   - ルールを削除
   - ルールを編集
4. ルールファイルに**変更を適用**

## インタラクション

```text
どのルールを設定しますか？
> 2

選択: warn-console-log

何をしますか？
1. 有効化（現在: ❌）
2. ルールを削除
3. パターンを編集
4. キャンセル

> 1

✅ ルール 'warn-console-log' が有効化されました。
```

## 手順

1. まず`/hookify:list`を実行して利用可能なルールを表示
2. AskUserQuestionを使用してルール選択を取得
3. 現在の状態に基づいて設定オプションを表示
4. 選択されたファイルのYAMLフロントマターを更新
5. ユーザーに変更を確認
