# /plans - 計画ドキュメントビューアー

計画ドキュメント（SOW/Spec）を一覧・表示。

## 入力

- 機能名: `$1`（任意、特定の機能を表示）
- `$1`が空の場合 → 全SOWドキュメントを一覧

## 実行

検索パス（プロジェクト固有を先に検索）:

```text
$HOME/.claude/workspace/planning/**/sow.md
```

各SOWから `Status:` 行の値を抽出。

### プランアクション（一覧表示後）

一覧表示 → AskUserQuestionでプラン番号を選択し詳細表示、またはスキップ。

## 出力

| 列         | ソース                                                         |
| ---------- | -------------------------------------------------------------- |
| #          | 連番                                                           |
| 場所       | "プロジェクト" (`.claude/`) or "グローバル" (`$HOME/.claude/`) |
| 機能       | ディレクトリ名（日付プレフィックス以降）                       |
| 日付       | ディレクトリ名プレフィックス (YYYY-MM-DD)                      |
| ステータス | sow.md の `Status:` フィールド値                               |

```markdown
## 利用可能なSOWドキュメント

| #   | 場所         | 機能          | 日付       | ステータス  |
| --- | ------------ | ------------- | ---------- | ----------- |
| 1   | プロジェクト | feature-name  | 2025-01-14 | in-progress |
| 2   | グローバル   | other-feature | 2025-01-13 | draft       |
```

### ステータス凡例

| ステータス  | 意味         |
| ----------- | ------------ |
| draft       | 計画フェーズ |
| in-progress | 実装開始済み |
| completed   | 全AC検証完了 |
