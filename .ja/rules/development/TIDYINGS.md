# Tidyings

**いつ**: メインタスク後、コミット前、編集したファイルのみ
**ルール**: 動作変更なし

## 許可

| カテゴリ     | アクション                                  |
| ------------ | ------------------------------------------- |
| 空白         | 末尾スペース、EOF、インデント               |
| インポート   | 未使用削除、アルファベット順、グループ化    |
| 変数         | 未使用削除、単一使用インライン化、let→const |
| コメント     | 解決済みTODO削除、デッドコード              |
| 型（TS）     | 推論可能削除、any→具体的                    |
| フォーマット | 一貫したセミコロン/クォート/カンマ          |
| 命名         | タイポ修正、ケース不一致修正                |

## 絶対にしない

ロジック、構造、機能、パフォーマンス、API変更、テスト修正

## レポート

`🧹 Tidyings: 空白 ✓ インポート ✓ 未使用 ✓`
