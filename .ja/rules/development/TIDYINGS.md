# Tidyings

| 範囲   | 説明                                             |
| ------ | ------------------------------------------------ |
| 適用時 | 編集したファイル限定でメインタスク後、コミット前 |
| ルール | 振る舞いを変えない                               |

## 許可

| カテゴリ     | アクション                                          |
| ------------ | --------------------------------------------------- |
| 空白         | 行末空白、EOF、インデント                           |
| import       | 未使用を削除し、アルファベット順、グループ化        |
| 変数         | 未使用を削除、単一使用は inline、let → const        |
| コメント     | 解決済み TODO の削除、デッドコードの削除            |
| 型 (TS)      | 推論可能なものを削除、any → 具体型                  |
| 整形         | セミコロン/クォート/カンマの統一                    |
| 命名         | typo 修正、ケースの不整合の修正                     |
| 一時ファイル | このタスクで iteration 用に作った一時ファイルを削除 |

## 不許可

ロジック、構造、機能、性能、API 変更、テスト修正。
