# Tidyings

| 範囲   | 説明                                             |
| ------ | ------------------------------------------------ |
| 適用時 | メインタスク後、コミット前。編集したファイル限定 |
| ルール | 振る舞いを変えない                               |

## 許可

| カテゴリ | アクション                                                             |
| -------- | ---------------------------------------------------------------------- |
| 空白     | 行末空白、EOF、インデント                                              |
| import   | 未使用 (このタスクで導入したもの) を削除、アルファベット順、グループ化 |
| 変数     | 未使用 (このタスクで導入したもの) を削除、単一使用は inline、let→const |
| コメント | 解決済み TODO の削除、デッドコード (このタスクで導入したもの) の削除   |
| 型 (TS)  | 推論可能なものを削除、any→具体型                                       |
| 整形     | セミコロン/クォート/カンマの統一                                       |
| 命名     | typo 修正、ケースの不整合の修正                                        |
| 一時ファイル | このタスクで iteration 用に作った scratch/helper ファイルを削除    |

## 不許可

- 既存のデッドコード (このタスク以前から存在するもの)。ユーザーに通知し、削除しない。
- ロジック、構造、機能、性能、API 変更、テスト修正。

## レポート

`🧹 Tidyings: Whitespace ✓ Imports ✓ Unused ✓`
