# パフォーマンスガイドライン

## コンテキスト管理

| 戦略       | ガイドライン                | 根拠                                 |
| ---------- | --------------------------- | ------------------------------------ |
| MCP tools  | プロジェクトごと ≤10 有効化 | 200k→70k コンテキスト縮小を防止      |
| Plugins    | 必要なものだけ有効化        | 各プラグインがトークンオーバーヘッド |
| `/compact` | コンテキスト >70% で使用    | 会話履歴を保持                       |
| `/fork`    | 並列タスクに使用            | コンテキスト汚染を回避               |
