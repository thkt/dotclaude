# 失敗パターン

AI が検出し、適切なコマンドを提案するためのパターン。

## 検出と提案

| パターン         | トリガー                       | 提案                           |
| ---------------- | ------------------------------ | ------------------------------ |
| コンテキスト膨張 | 無関係なトピック / 使用率 >70% | `/clear` または `/compact`     |
| 繰り返し修正     | 同じエラーで3回目の試行        | より具体的にプロンプトを再構成 |
| 無限探索         | >10 ファイル読み込み、編集なし | subagent でスコープを絞る      |
| 間違った方向     | 「それは望んでいたものと違う」 | `/rewind` でチェックポイントへ |
| 長い待機         | ユーザーが操作中に待機         | `Esc` で中断を提案             |

検出時: 通知 → 提案 → 確認を待つ
