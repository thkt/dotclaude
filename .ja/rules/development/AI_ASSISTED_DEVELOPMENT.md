# AI支援開発

**マインドセット**: AIが生成、人間が検証

## プロジェクトルール

1. **レビューは必須** - AI生成コードは理解してからコミット
2. **過去の例 > 抽象ルール** - 既存コードを例として参照
3. **コードはあなたのもの** - AIはアシスタント、代替ではない

## 統合

| 原則                           | AIの役割               | 人間の役割         |
| ------------------------------ | ---------------------- | ------------------ |
| TDD                            | テスト雛形を生成       | アサーションを記述 |
| オッカムの剃刀                 | すべての抽象化を疑問視 | シンプルさを適用   |
| プログレッシブエンハンスメント | シンプル版を生成       | 必要に応じて拡張   |
