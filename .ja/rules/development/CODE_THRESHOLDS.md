# Code Thresholds

機械的強制: oxlint (`max-lines-per-function`, `max-lines`, `max-depth`,
`max-params`, `complexity`) — guardrails hook 経由

| 対象                     | 推奨 | 最大 (oxlint error) |
| ------------------------ | ---- | ------------------- |
| 関数行数                 | ≤30  | 50                  |
| ファイル行数             | ≤400 | 800                 |
| ネスト深度               | ≤3   | 4                   |
| 関数引数                 | ≤3   | 5                   |
| サイクロマティック複雑度 | ≤10  | 15                  |

## カバレッジ

| レベル | 目標 | 対象       |
| ------ | ---- | ---------- |
| C0     | ≥90% | 全行実行   |
| C1     | ≥80% | 全分岐実行 |

例外: 自動生成コード、データ定義ファイル、テストファイル、移行中のレガシー
