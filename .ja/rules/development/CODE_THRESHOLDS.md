# Code Thresholds

これらの推奨値を目指す。ハードリミットは oxlint（guardrails hook 経由）で強制。

| 対象                     | 推奨 |
| ------------------------ | ---- |
| 関数行数                 | ≤30  |
| ファイル行数             | ≤400 |
| ネスト深度               | ≤3   |
| 関数引数                 | ≤3   |
| サイクロマティック複雑度 | ≤10  |

## カバレッジ

| レベル | 目標 | 対象       |
| ------ | ---- | ---------- |
| C0     | ≥90% | 全行実行   |
| C1     | ≥80% | 全分岐実行 |

例外: 自動生成コード、データ定義ファイル、テストファイル、移行中のレガシー
