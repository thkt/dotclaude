# Code Thresholds

## 閾値一覧

| 対象                     | 推奨 | 警告    | 最大 | 根拠          |
| ------------------------ | ---- | ------- | ---- | ------------- |
| 関数行数                 | ≤30  | 31-50   | 50   | Clean Code    |
| ファイル行数             | ≤400 | 401-800 | 800  | Code Complete |
| ネスト深度               | ≤3   | 4       | 4    | Miller's Law  |
| 関数引数                 | ≤3   | 4-5     | 5    | Clean Code    |
| サイクロマティック複雑度 | ≤10  | 11-15   | 15   | McCabe        |

## 重大度

| 違反            | 重大度 | 対応     |
| --------------- | ------ | -------- |
| 関数 >50行      | High   | 必須修正 |
| ファイル >800行 | High   | 必須修正 |
| ネスト >4段     | Medium | 推奨修正 |
| 引数 >5個       | Medium | 推奨修正 |
| 複雑度 >15      | Medium | 推奨修正 |

## 例外

| ケース             | 理由                       |
| ------------------ | -------------------------- |
| 自動生成コード     | 手動編集しない             |
| データ定義ファイル | 構造的に長くなる           |
| テストファイル     | 独立したテストケースの集合 |
| 移行中のレガシー   | 段階的リファクタリング対象 |
