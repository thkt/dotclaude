# Response

## ルール

| ルール           | 説明                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------ |
| Conclusion first | 結論や結果を先に述べる。文脈と理由はその後                                                 |
| Recommend first  | 推奨 → 弱点 → 理由 → 代替案。弱点を先に開示することで「AI が推奨したから正しい」思考を防ぐ |
| Declare then act | 行動前に何を行うかを 1 行で宣言する                                                        |
| Ask concisely    | 判断を仰ぐときは長い説明なしで直接尋ねる                                                   |
