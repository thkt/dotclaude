# テンプレート変数

テンプレートとコマンド出力での変数置換構文。

## 構文

| パターン                          | 入力                          | 出力           | 説明               |
| --------------------------------- | ----------------------------- | -------------- | ------------------ |
| `{field}`                         | `name: MyApp`                 | `MyApp`        | 単純フィールド     |
| `{object.property}`               | `summary: {total: 8}`         | `8`            | ネスト             |
| `{array[].property}`              | `items: [{id: 1}, {id: 2}]`   | `1`, `2`       | 配列イテレーション |
| `{array[filter=value].property}`  | `list: [{p: high}, {p: low}]` | 最初の一致     | フィルタ (等値)    |
| `{array[filter!=value].property}` | `list: [{p: high}, {p: low}]` | 一致しない全件 | フィルタ (除外)    |

## エッジケース

| 入力                 | 出力           |
| -------------------- | -------------- |
| 空配列               | 何も出力しない |
| 存在しないプロパティ | 空文字列       |
