# プラグイン規約

`.claude/.claude-plugin/` 配下の Claude Code プラグイン定義のルール。

## 制約

| ルール             | ガイドライン                                      |
| ------------------ | ------------------------------------------------- |
| モノリシック構成   | `marketplace.json` では `source: "./"` を使う     |
| 相互参照の保持     | skills/, rules/, agents/ 間の相互参照を破壊しない |
| 外部プラグイン禁止 | 外部ホスティングされたプラグインに分割しない      |

## 理由

プラグインはロード時にキャッシュされ、境界を越えたファイル参照ができない。外部ホスティングは `skills/`, `rules/`, `agents/` 間の既存の相互参照を破壊する。
