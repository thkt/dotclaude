# AI文章トロープ回避（日本語版）

対象: PR説明、Issue、ドキュメント、ブログ等の日本語散文。
対象外: コード、コードコメント、技術テーブル、コミットメッセージ。

1つなら許容。複数重なるとAIスロップ。

## 語彙

| トロープ     | 日本語での例                                 | 修正                         |
| ------------ | -------------------------------------------- | ---------------------------- |
| 魔法の副詞   | 本質的に、根本的に、極めて、まさに           | 削除か具体表現に             |
| AI頻出動詞   | 深掘りする、活用する、推進する、実現する     | 平易な動詞に                 |
| 大仰な名詞   | 地平、様相、全体像、エコシステム、パラダイム | 具体名詞に                   |
| 迂回コピュラ | 〜として機能する、〜の役割を果たす、〜を担う | 「〜である」で済むなら素直に |
| 体言止め乱用 | 圧倒的な速度。革新的なアプローチ。           | 述語を省略しすぎない         |

## 構文

| トロープ           | 日本語でのパターン                                   | 修正               |
| ------------------ | ---------------------------------------------------- | ------------------ |
| 否定対比構文       | 「〜ではない。〜なのだ。」「〜ではなく〜である」     | 肯定文で直接述べる |
| 劇的カウントダウン | 「〜でもない。〜でもない。〜なのだ。」               | 結論を直接述べる   |
| 自問自答           | 「では〜とは何か？それは〜である。」                 | 1文にまとめる      |
| 文頭反復           | 「〜が必要です。〜が必要です。〜が必要です。」       | 文頭を変える       |
| 三連発             | 同構造の3つ組を連続使用                              | 1回だけ            |
| 空フィラー         | 特筆すべきは、注目すべきは、興味深いことに           | 削除               |
| 浅い分析の付け足し | 「〜の重要性を示している」「〜を浮き彫りにしている」 | 削除か具体化       |
| 偽レンジ           | 「〜から〜に至るまで」（実際にスケール上にない）     | 実際の範囲のみ     |

## 段落・構成

| トロープ               | 日本語でのパターン                               | 修正                 |
| ---------------------- | ------------------------------------------------ | -------------------- |
| 短文断片の乱発         | 一語・一句だけの段落を連発                       | 段落にまとめる       |
| 偽装リスト             | 「第一に〜。第二に〜。第三に〜。」を散文に混ぜる | リストか散文か統一   |
| フラクタル要約         | セクション末＋全体末で同内容を繰り返す           | 要約は1回まで        |
| メタファー固執         | 同じ比喩を記事全体で5回以上                      | 1回使って次へ        |
| 歴史アナロジー連発     | 「Appleは〜。Facebookは〜。Stripeは〜。」        | 1例まで              |
| 一論点の水増し         | 同じ主張を表現を変えて繰り返す                   | 1回で言い切る        |
| 結論宣言               | まとめると、以上のように、総括すると             | 宣言せず自然に閉じる |
| 課題楽観パターン       | 「課題はあるものの〜」で全て楽観に回収           | 課題に向き合うか省略 |
| 「〜と言えるでしょう」 | 断言回避の過剰ヘッジ                             | 根拠があるなら断言   |

## トーン

| トロープ         | 日本語でのパターン                                 | 修正                     |
| ---------------- | -------------------------------------------------- | ------------------------ |
| 偽サスペンス     | 「ここで重要なのは」「実はここに落とし穴がある」   | 要点を直接述べる         |
| 上から目線の喩え | 「いわば〜のようなものです」の連発                 | 読者を信頼する           |
| 未来誘導         | 「〜な世界を想像してみてください」                 | 具体的な提案を書く       |
| 偽の謙虚さ       | 「もちろん筆者の見解に過ぎませんが」の定型挿入     | 具体的か不要なら省略     |
| 自明の主張       | 「言うまでもなく」「明らかに」「自明ですが」       | 証明するか省略           |
| 重大性の水増し   | 全てを歴史的転換点にする                           | 実際のスケールに合わせる |
| 教師モード       | 「順を追って見ていきましょう」「整理しましょう」   | そのまま説明する         |
| 曖昧な出典       | 「専門家によると」「多くの報告が示すように」       | 出典を明記するか削除     |
| 造語ラベル       | 「〜パラドックス」「〜ジレンマ」を既存概念のように | 定義するか平易な表現に   |
| 過剰な接続表現   | 「一方で」「他方」「翻って」の連発                 | 必要な箇所だけ           |

| トロープ         | パターン                               | 修正                         |

| ダッシュ中毒     | ダッシュ（――）を1記事で大量使用        | 2-3箇所まで                  |
