# ドキュメントルール

Claude Codeエコシステムにおける一貫性のあるドキュメントのガイドライン。

## コア原則

1. **完全性より明確性** - オッカムの剃刀を適用
2. **EN/JP同期** - 両版の構造は一致が必須
3. **循環参照なし** - 最大3レベルまで
4. **ASCIIよりMermaid** - ASCII図ではなくMermaidダイアグラムを使用

## ディレクトリ構造

```text
/rules/
├── core/           # AI動作（hook注入）
├── conventions/    # ドキュメント規約
├── development/    # 実践的手法
└── workflows/      # ワークフローガイド & モジュール化ルール

/skills/            # ナレッジベース（SOLID, DRY等）
```

## 参照ルール

| ルール     | ガイドライン                                    |
| ---------- | ----------------------------------------------- |
| 最大深度   | 3レベル                                         |
| 循環参照   | 禁止（A → B → A）                               |
| 冗長な参照 | CLAUDE.mdで既にロードされるファイルを参照しない |
| 必要時参照 | 現在のコンテキストで必要な場合のみ参照          |

## 言語同期

[@./TRANSLATION.md](./TRANSLATION.md) を参照

## 更新チェックリスト

| 操作 | ステップ                          |
| ---- | --------------------------------- |
| 追加 | EN+JP作成 → CLAUDE.mdに追加       |
| 変更 | grep参照 → EN+JP更新 → リンク確認 |
| 移動 | 参照検索 → EN+JP移動 → 参照更新   |

## 記述標準

- ヘッダー: 文頭大文字
- 例: 悪い vs 良い の比較
- コメント: AI文書にはテキストラベル、人間出力には絵文字OK

## フォーマットルール

**テンプレート変数** - [@./TEMPLATE_VARIABLES.md](./TEMPLATE_VARIABLES.md) を参照

**太字（`**text**`）** - 明確な意図がある場合のみ使用：

| 有効な使用    | 例                             | 目的           |
| ------------- | ------------------------------ | -------------- |
| ルール/原則名 | `**Safety First**`             | 定義の明確化   |
| 重要な指示    | `**Never proceed**`            | アクション強調 |
| ラベル        | `**When**:`, `**Guidelines**:` | 構造の明示     |
| 対比ペア      | `**using**` vs `**creating**`  | 区別           |

**無効な使用**: 目的のない任意の強調（例：新項目のハイライト）

## 簡潔化ガイドライン

| 原則               | 適用                                                              |
| ------------------ | ----------------------------------------------------------------- |
| DRY                | 同じ情報は1箇所のみ（Metrics + Checklist + Rules に重複させない） |
| 閾値のインライン化 | 閾値はルールに直接含める、別テーブルにしない                      |
| 最小限の例         | パターンを示す、完全な実装は不要                                  |
| 冗長なコメント不要 | 他で説明済みならコードブロック内で繰り返さない                    |
