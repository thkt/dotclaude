# プラグイン構造

Claude Codeプラグインアーキテクチャを維持するためのルール。

## ルール

1. **モノリシック維持**: marketplace.jsonのすべてのプラグインで `source: "./"` を維持
2. **相互参照保持**: skills/、rules/、agents/ 間の相互参照を破壊しない
3. **外部共有制限**: 個別プラグインの外部化を試みない

## 根拠

ADR 0003から抽出。Claude Codeプラグインキャッシュ制約により:

- **プラグインキャッシュ制限**: プラグインシステムはロード時にキャッシュを行い、プラグイン境界を越えた `shared/` ディレクトリアクセスを防止
- **参照整合性**: コンポーネント間に394以上の相互参照が存在。これを破壊すると広範な障害が発生
- **安定性**: モノリシック構造は既存ワークフロー（/think, /code, /audit, /commit）で安定動作実績あり

## 適用タイミング

| シナリオ                         | アクション              |
| -------------------------------- | ----------------------- |
| 新しいプラグイン追加             | `source: "./"` を使用   |
| 外部共有を検討                   | **しない** - 制限を説明 |
| プラグイン構造のリファクタリング | すべての相互参照を保持  |
| コンポーネント間のファイル移動   | すべての参照を更新      |

## 構造

```text
.claude/
├── .claude-plugin/
│   └── marketplace.json    # すべてのプラグインは source: "./"
├── skills/                 # 教育的コンテンツ
├── rules/                  # 強制ガイドライン
├── agents/                 # 特化エージェント
├── commands/               # ユーザー向けコマンド
└── [上記すべての間の相互参照]
```

## marketplace.json パターン

```json
{
  "plugins": [
    {
      "name": "core-rules",
      "source": "./" // 常にローカル
    },
    {
      "name": "development-skills",
      "source": "./" // 常にローカル
    }
  ]
}
```

## 例外

以下の場合にこのルールを再検討可能:

1. **公式プラグインシステム更新**: Claude Codeが `shared/` ディレクトリサポートを追加
2. **新しい分離メカニズム**: プラグイン境界を越えてリソースを共有する方法が提供される
3. **代替アーキテクチャ**: 参照を破壊せずに外部共有を可能にするパターンが登場

それまではモノリシック構造を維持。

## アンチパターン

### Bad: プラグイン分離を試みる

```json
{
  "plugins": [
    {
      "name": "core-rules",
      "source": "git://example.com/rules.git" // 動作しない
    }
  ]
}
```

**問題点**:

- skills/ への相互参照が失敗
- プラグインキャッシュに共有リソースが含まれない
- ワークフローが破壊される

### Bad: 相互参照を破壊する

```text
Before:
  skills/tdd/ → rules/development/

After (破壊):
  plugins/skills/tdd/ → ??? (参照破壊)
```

## チェックリスト

プラグイン構造を変更する前に:

- [ ] すべての相互参照が有効のまま
- [ ] source: "./" が維持されている
- [ ] 既存ワークフローが正常に機能
- [ ] 外部プラグイン依存関係が追加されていない

## 検証

```bash
# 破壊された参照をチェック
rg "@\.\.\/" --glob "*.md" | wc -l

# marketplace.jsonを検証
jq '.plugins[].source' .claude-plugin/marketplace.json
# すべて "./" を返すべき
```
