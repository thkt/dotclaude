# 空句 (日本語)

Issue 本文に出やすい LLM 的な空句を検出して直す。本文が日本語のとき prose-review.md と併せて使う。

| パターン         | シグナル                                              | 修正                               |
| ---------------- | ----------------------------------------------------- | ---------------------------------- |
| 定型の書き出し   | 「この Issue は〜を説明する / 報告する / 提案する」   | 自己紹介でなく問題から始める       |
| 空虚な強調語     | 不可欠 / 抜本的 / 包括的                              | 削除するか件数や名前の具体に置換 |
| あいまいな数量詞 | いくつかの問題 / 複数の懸念 / さまざまなバグ          | 列挙するか件数を示す               |
| ヘッジの重ね掛け | おそらく〜かもしれない / 〜の可能性があるかもしれない | hedge は最大 1 つ、または断言する  |
| 埋め草フレーズ   | 「ご意見をお待ちしています」「フィードバック歓迎」    | 削除。事実を述べるか直接尋ねる     |
