# Flaky テスト管理

## 戦略

| フェーズ | アクション       | パターン/コマンド        |
| -------- | ---------------- | ------------------------ |
| Tag      | flaky をマーク   | `describe("@flaky ...")` |
| Isolate  | CI から除外      | `--grep-invert @flaky`   |
| Debug    | flaky のみ実行   | `--grep @flaky`          |
| Verify   | 連続 5 回 pass   | `--repeat-each=5`        |
| Promote  | tag を外し、監視 | CI の安定性チェック      |

## 原因カテゴリ

| カテゴリ | 例                                     |
| -------- | -------------------------------------- |
| Timing   | 非同期、アニメーション、API レイテンシ |
| Order    | 共有状態、並列衝突                     |
| External | ネットワーク、時刻、環境               |

## メトリクス

| メトリック     | 健全   | 警告    |
| -------------- | ------ | ------- |
| Flaky 率       | < 1%   | > 5%    |
| 修正までの日数 | < 7 日 | > 14 日 |
| 再発率         | < 10%  | > 20%   |
