# Flaky Test Management

## 戦略

| フェーズ | アクション         | パターン/コマンド        |
| -------- | ------------------ | ------------------------ |
| Tag      | フレーキーをマーク | `describe("@flaky ...")` |
| Isolate  | CIから除外         | `--grep-invert @flaky`   |
| Debug    | フレーキーのみ実行 | `--grep @flaky`          |
| Verify   | 5回連続成功        | `--repeat-each=5`        |
| Promote  | タグ削除、監視     | CI安定性チェック         |

## 原因カテゴリ

| カテゴリ | 例                              |
| -------- | ------------------------------- |
| Timing   | 非同期、アニメーション、API遅延 |
| Order    | 共有状態、並列競合              |
| External | ネットワーク、時間、環境        |

## メトリクス

| 指標           | 健全  | 警告   |
| -------------- | ----- | ------ |
| フレーキー率   | < 1%  | > 5%   |
| 修正までの日数 | < 7日 | > 14日 |
| 再発率         | < 10% | > 20%  |
