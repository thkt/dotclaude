# setting-up-docsの評価

## 選択基準

このスキルをトリガーするキーワードとコンテキスト:

- **キーワード**: setup guide, environment setup, development environment, installation guide, getting started, prerequisites, セットアップガイド, 環境構築, 開発環境, インストールガイド, 始め方, 前提条件
- **コンテキスト**: 環境ドキュメント、オンボーディングガイド、/docs:setupコマンド

## 評価シナリオ

### シナリオ1: 環境セットアップガイド生成

```json
{
  "skills": ["setting-up-docs"],
  "query": "このプロジェクトのセットアップガイドを自動生成して",
  "files": [],
  "expected_behavior": [
    "スキルが'セットアップガイド'でトリガーされる",
    "パッケージマネージャを検出（npm/yarn/pnpm/pip）",
    "必要なツールとバージョンをリスト",
    ".env.exampleから環境変数を抽出",
    "起動コマンドをドキュメント化"
  ]
}
```

### シナリオ2: 前提条件の検出

```json
{
  "skills": ["setting-up-docs"],
  "query": "開発に必要な前提条件をドキュメント化して",
  "files": [],
  "expected_behavior": [
    "スキルが'前提条件'でトリガーされる",
    ".nvmrcからNode.jsバージョンを検出",
    ".python-versionからPythonバージョンを検出",
    "asdfからtool-versionsをリスト",
    "必要なグローバルインストールを示す"
  ]
}
```

### シナリオ3: 環境変数ドキュメント

```json
{
  "skills": ["setting-up-docs"],
  "query": "必要な環境変数をドキュメント化したい",
  "files": [".env.example"],
  "expected_behavior": [
    "スキルが'環境変数'でトリガーされる",
    ".env.exampleファイルをパース",
    "必要なすべての変数をリスト",
    "コメントから説明を含める",
    "デフォルト値があれば示す"
  ]
}
```

### シナリオ4: マルチ言語プロジェクトセットアップ

```json
{
  "skills": ["setting-up-docs"],
  "query": "モノレポの各パッケージのセットアップ手順を出して",
  "files": [],
  "expected_behavior": [
    "スキルが'セットアップ'でトリガーされる",
    "複数のパッケージマネージャを検出",
    "パッケージごとのセットアップ手順を生成",
    "ワークスペースレベルのコマンドを示す",
    "パッケージ間依存関係をドキュメント化"
  ]
}
```

### シナリオ5: Docker開発環境

```json
{
  "skills": ["setting-up-docs"],
  "query": "Docker開発環境のセットアップをドキュメント化して",
  "files": ["docker-compose.yml", "Dockerfile"],

    "スキルが'Docker'と'セットアップ'でトリガーされる",

    "サービスとポートをドキュメント化",
    "必要なdockerコマンドを示す",
    "ボリュームマウントとネットワークをリスト"
  ]

```

## 手動検証チェックリスト

各シナリオ実行後:

- [ ] スキルがセットアップキーワードで正しくトリガーされた
- [ ] パッケージマネージャが正しく検出された
- [ ] 必要なツールとバージョンがリストされた
- [ ] 環境変数がドキュメント化された
- [ ] 起動コマンドが正確だった
- [ ] マルチ言語/モノレポサポートが機能した

## ベースライン比較

### スキルなし

- 手動でREADME作成
- 必要なツールを見逃す可能性
- 古いバージョン要件
- 不完全な環境変数ドキュメント

### スキルあり

- 設定ファイルからの自動検出
- マルチパッケージマネージャサポート
- .nvmrc等からのバージョン検出
- 環境変数の抽出
- Dockerセットアップドキュメント
