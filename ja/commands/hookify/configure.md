# /hookify:configure - フックルール設定

フックルールを対話的に有効化、無効化、または削除します。

## プロセス

1. **ルール一覧表示**: `/hookify:list`で表示
2. **ユーザーに質問**: どのルールを設定するか
3. **オプション表示**:
   - ルールを有効化
   - ルールを無効化
   - ルールを削除
   - ルールを編集
4. **変更を適用**: ルールファイルに適用

## 対話例

```text
どのルールを設定しますか？
> 2

選択: warn-console-log

何をしますか？
1. 有効化（現在: ❌）
2. ルールを削除
3. パターンを編集
4. キャンセル

> 1

✅ ルール 'warn-console-log' が有効化されました。
```

## 手順

1. まず`/hookify:list`を実行して利用可能なルールを表示
2. AskUserQuestionを使用してルール選択を取得
3. 現在の状態に基づいて設定オプションを表示
4. 選択されたファイルのYAMLフロントマターを更新
5. ユーザーに変更を確認
